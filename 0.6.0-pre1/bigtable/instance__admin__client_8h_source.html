44 /**
45  * Connects to Cloud Bigtable's instance administration APIs.
46  *
47  * This class is used by the Cloud Bigtable wrappers to access Cloud Bigtable.
48  * Multiple `bigtable::InstanceAdmin` objects may share a connection via a
49  * single `InstanceAdminClient` object. The `InstanceAdminClient` object is
50  * configured at construction time, this configuration includes the credentials,
51  * access endpoints, default timeouts, and other gRPC configuration options.
52  * This is an interface class because it is also used as a dependency injection
53  * point in some of the tests.
54  *
55  * @par Cost
56  * Applications should avoid unnecessarily creating new objects of type
57  * `InstanceAdminClient`. Creating a new object of this type typically requires
58  * connecting to the Cloud Bigtable servers, and performing the authentication
59  * workflows with Google Cloud Platform. These operations can take many
60  * milliseconds, therefore applications should try to reuse the same
61  * `InstanceAdminClient` instances when possible.
62  */
