25  * Define the interfaces to create ReadWriteModifyRule operations.
26  *

28  * incrementing an integer value or appending to a string value. The changes are
29  * represented by a ReadModifyWriteRule operations. One or much such operations

31  * through a more idiomatic C++ interface.
32  */
