Detailed Description

Disable MD5 Hashing computations.

By default the GCS client library computes MD5 hashes in all Client::InsertObject calls. The application can disable the hash computation by passing this parameter.

Warning
Disabling MD5 hashing exposes your application to data corruption. We recommend that all uploads to GCS are protected by the supported checksums and/or hashes.
