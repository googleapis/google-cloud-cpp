ListObjectsReader::iterator google::cloud::storage::v0::ListObjectsReader::begin ( )

Return an iterator over the list of objects.

The returned iterator is a single-pass input iterator that reads object metadata from the ListObjectsReader when incremented.

Creating, and particularly incrementing, multiple iterators on the same ListObjectsReader is unsupported and can produce incorrect results.
